2. Les Installeurs

a) Unity

1. Ouvrir l'installeur d'Unity et cliquer sur "next"

2. Accepter et cliquer sur "next"

3. Selectionner votre type de systeme puis cliquer sur "next"

4. Tout décocher sauf "Unity NUMERO DE VERSION" et cliquer sur "next"

5. Sélectionner "Download To", entrer un chemin de téléchargement,

dans "Unity install folder" entrer "C:\Unity" et cliquer sur "next"

6. Attendre la fin du téléchargement, c'est long ...

b) GitHub For Desktop

1. Ouvrir l'installeur de GitHub For Desktop et cliquer sur "installer"

2. Attendre la fin du téléchargement ...

c) Visual Studio 2015

1. Ouvrir l'installeur d'Unity et cliquer sur "next"

2. Accepter et cliquer sur "next"

3. Selectionner votre type de systeme puis cliquer sur "next"

4. Tout décocher sauf "Visual Studio" et cliquer sur "next"

5. Sélectionner "Download To", entrer un chemin de téléchargement,

dans "Unity install folder" entrer "C:\Unity" et cliquer sur "next"

6. Attendre la fin du téléchargement, c'est pas long

7. Puis suivre les indication
